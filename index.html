Driver Information API

How to Access Driver Data

To access the data of a specific driver, use the following URL format:

https://praneeth7781.github.io/f1nsight-api-2/drivers/driver_id.json

Replace driver_id with the actual ID of the driver you want to retrieve information about.

Example

To get information about the driver with ID driver1, you would use the following URL:

https://praneeth7781.github.io/f1nsight-api-2/drivers/driver1.json

How to Access Year-Wise Constructors

To access the list of constructors participating/participated in a particular year, use the following URL format:

https://praneeth7781.github.io/f1nsight-api-2/constructors/year.json

Replace year with the actual year

Example

To get constructors participating in the year 2023, you would use the following URL:

https://praneeth7781.github.io/f1nsight-api-2/constructors/2023.json

How to Access Constructor Colors

To access the colors of teams in JSON format with team names as keys, use the following URL

https://praneeth7781.github.io/f1nsight-api-2/colors/teams.json

To access the colors of teams in JSON format with driver IDs as keys, use the following URL

https://praneeth7781.github.io/f1nsight-api-2/colors/drivers.json
