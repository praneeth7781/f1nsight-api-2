Driver Information API

How to Access Driver Data

To access the data of a specific driver, use the following URL format:

https://praneeth7781.github.io/drivers/driver_id.json

Replace driver_id with the actual ID of the driver you want to retrieve information about.

Example

To get information about the driver with ID driver1, you would use the following URL:

https://praneeth7781.github.io/drivers/driver1.json
